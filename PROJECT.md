# Lens By Steven - Projet Photographe Professionnel

## Vision
Un site portfolio premium, minimaliste et performant pour un photographe professionnel (Steven). L'objectif est de mettre en avant la qualité visuelle des travaux (Portraits, Sport, Entreprises, Événementiel) tout en offrant une expérience utilisateur fluide et luxueuse.

## Stack Technique
- **Framework**: Next.js 14 (App Router)
- **Langage**: TypeScript
- **Styling**: Tailwind CSS / Vanilla CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Skills Intégrés**: 
  - `gsd`: Pour la gestion de projet et la qualité.
  - `ui-ux-pro-max`: Pour l'intelligence design et les interactions premium.
  - `awesome-claude`: Pour l'accès à des outils spécialisés.

## État Actuel
- Catégorisation du portfolio finalisée (v1.0).
- Structure de base (Hero, Services, About, Testimonials, Contact) en place.
- Design responsive OK.

## Objectifs v2.0
1. **Élévation UI/UX**: Micro-interactions avancées, transitions de pages fluides.
2. **Performance**: Optimisation du chargement des images (Next/Image + blur data).
3. **Fonctionnalités**: Système de réservation (booking) ou galeries privées clients.
4. **SEO & Accessibilité**: Audit complet via les ressources `awesome-claude`.
